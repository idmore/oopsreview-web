## Getting Started
 - ### Development / Local
    - **run server**
      ```
      yarn start
      ```
    - **webpack watch**
      ```
      yarn webpack:watch
      ```
- ### Production
  - **build**
    ```
    yarn build:prod
    ```  
  - **run server**
    ```
    dist/server/index.js
    ```
